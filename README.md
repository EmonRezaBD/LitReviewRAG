# Sample Papers

This directory holds PDFs of research papers used to demonstrate the pipeline.

**PDFs are not committed** to the repository to respect copyright. To reproduce
the demo, download 2–3 open-access papers (e.g., from [arXiv](https://arxiv.org))
on any topic and place them here as `sample1.pdf`, `sample2.pdf`, etc.

Suggested papers (open-access):
- Park et al., "Generative Agents: Interactive Simulacra of Human Behavior" — https://arxiv.org/abs/2304.03442
- Park et al., "Generative Agent Simulations of 1,000 People" — https://arxiv.org/abs/2411.10109
